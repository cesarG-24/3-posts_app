# React App.

### Fetch data from API.

---

This application consists of obtaining the API data and with this data create:

- Post:
    * Each post must contain the user who created the post (Full name, profile pic.),
    * post title.
    * post body.
    * comments.
- show 16 posts per page.
- Use pagination.
